## Features ✨
- 📸 **Screenshot Capture**: Select screen regions with intuitive overlay
- 🖼️ **Multi-Format Support**: Process PNG, JPG, BMP, TIFF, WEBP
- 🔍 **Advanced OCR**: Powered by Tesseract with preprocessing
- 📋 **Export Options**: Save as TXT/DOCX or copy to clipboard
- 🖥️ **Modern GUI**: User-friendly interface with status updates
- ⚙️ **Auto-Save**: Automatically archives screenshots and extractions

## Installation 🛠️
### 1. Install Tesseract OCR
- **Windows**: [Download installer](https://github.com/UB-Mannheim/tesseract/wiki)
- **Linux**: `sudo apt install tesseract-ocr libtesseract-dev`
- **Mac**: `brew install tesseract`
